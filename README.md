## Weather App (Python)

A simple Python script that retrieves current weather data for a given city using the [OpenWeatherMap API](https://openweathermap.org/api).

---

## Features

- Get current temperature in Celsius
- Get current humidity
- Simple command-line interface
- Easy to set up and use

---

## Prerequisites

- Python 3.6 or higher
- `requests` library
